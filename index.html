Dashboard 03 · The Automated One

Daily Pricing — Automated Intelligence Pipeline

Open Dashboard ↗
Scroll ↓

Petrochemical prices are volatile. Without real-time pricing, teams risk quoting stale prices — losing margin or missing deals. This dashboard solves that through a fully automated pipeline — drop in a new monthly sheet and everything flows automatically.

The Dashboard
Daily Pricing Monitor
The Raw Excel Source
Before · Raw Data

A new sheet every month — "Mar Average", "Apr Average", and so on.

Previously, someone had to open each sheet, copy the numbers, clean up format errors, and paste them into a consolidated report. With this pipeline — zero manual work. Power Query detects the new sheet, parses it, and integrates it into the dataset automatically.

Inside the Automation

Eight pieces of logic that make this run itself.

01
Automatic Sheet Detection
Text.Contains([Name], "Average")

What it does: Automatically detects any new monthly sheet (e.g. "Mar Average") — no code changes needed when a new month arrives.

02
Dynamic Month Extraction
Text.BeforeDelimiter([Name], " ")

What it does: Reads the month name directly from the sheet name — the dataset always knows which month each price belongs to.

03
Transformation Pipeline
fn_TransformDaily([SheetData], [MonthName], [MonthNumber])

What it does: A single function converts messy Excel sheets into a structured daily dataset — consistent format every time.

04
Data Cleaning
Table.SelectRows(...)

What it does: Removes empty rows, invalid entries, and zero values before they reach the dashboard — no bad data slips through.

05
Daily Change Calculation
([Price] - [PrevPrice]) / [PrevPrice]

What it does: Computes day-over-day price movement automatically — users instantly see what went up and what went down.

06
Product Classification
Text.Contains([Product], ...)

What it does: Sorts every product into its proper category — PX, PTA, Naphtha, Benzene, Ethylene, Spread, Crude — with zero manual tagging.

07
Monthly Average
List.Average(...)

What it does: Calculates the monthly average price automatically — no more end-of-month spreadsheet formulas.

08
Product Mapping
Products = { Column → Product → Category → Unit }

What it does: Maps raw Excel columns into structured business data — a single source of truth for product names, categories, and units.

Learnings

Challenge & Response

⚠ Challenges Faced

📋
Inconsistent Data Formats

Different teams used different date formats, column names, and unit representations across Excel files — consolidation was impossible without manual fixing.

🕐
Manual Reporting Inefficiency

Weekly reports required 2–4 hours of manual compilation — prone to errors and always one step behind current business reality.

🔁
Data Duplication & Silos

Multiple versions of the same data existed across departments, causing confusion and contradicting reports — leading to distrust in numbers.

✓ How We Responded

📐
Standardized Excel Input Templates

Designed locked templates with data validation, standardized column headers, and enforced date/unit formats — eliminating inconsistency at the source.

⚡
Automated Power BI Dashboards

Replaced manual Excel with dashboards that refresh daily — reducing reporting effort from hours to zero, while improving accuracy and timeliness.

🔧
Power Query Data Cleaning

Built reusable Power Query scripts to detect duplicates, normalize formats, and merge data from all sources into a single verified master dataset.
